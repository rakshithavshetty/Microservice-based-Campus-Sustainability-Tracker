# Campus Sustainability Tracker UI

A modern web dashboard for tracking and promoting sustainability efforts on campus. Built with React, this project helps visualize environmental data, track events, and encourage eco-friendly actions among students and staff.

## User Interface
The UI features a clean, modern dashboard layout with a sidebar for easy navigation between sections:
- **Sidebar:** Quick access to Reports, Events, Rewards, Resources, and Account sections.
- **Dashboard Home:** Personalized greeting, motivational eco-facts, and a visually engaging layout.
- **Reports:** Tabbed charts for carbon emissions, water, energy consumption, and generation, with calendar/month pickers and ML insights.
- **Events:** Scrollable, styled event list with date formatting and icons.
- **Account:** Card-style account info with a visually distinct logout button.
- **Rewards & Resources:** Dedicated sections for future expansion.
- **Responsive Design:** Optimized for both desktop and mobile devices.

## Getting Started

### Prerequisites
- Node.js (v14 or higher recommended)
- npm

### Installation
```bash
npm install
```

### Running Locally
```bash
npm start
```
Visit [http://localhost:3000](http://localhost:3000) in your browser.

### Building for Production
```bash
npm run build
```

## Folder Structure
```
public/           # Static assets
src/
  components/     # Reusable UI components
  context/        # React Context providers
  pages/          # Main app pages (Dashboard, Reports, Events, etc.)
  services/       # API and data services
  styles/         # CSS Modules
```
